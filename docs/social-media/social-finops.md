## ROI Formula

```
Social Media ROI = (Revenue from social-attributed demos - Total cost) / Total cost x 100

Where:
- Revenue from social-attributed demos = # demos from social x close rate x avg deal value
- Total cost = Tool costs + Paid promotion + (Hours invested x hourly rate)
- Hourly rate estimate: ₹5,000/hr (founder time)
```
